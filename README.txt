Progama que lê uma sequência de DNA de um arquivo de texto e compara com a base de dados (também em arquivo .txt) para saber a quem o DNA analisado corresponde.

A base de dados deve ter o seguinte formato, onde a 1ª coluna corresponde ao nome e as demais aos Short Tandem Repeats (STRs):

name,AGATC,AATG,TATC
Alice,2,8,3
Bob,4,1,5
Charlie,3,2,5
